Markdown¶

Naast restructured text kun je ook markdown gebruiken voor het opmaken van tekst. Enkele voorbeelden:

  • dit is een lijst
  • met enkele elementen
  • in elk geval drie.

Je kunt ook werken met genummerde lijsten:

  1. één
  2. twee
  3. drie

Tekst kan cursief of vet of eventueel beide.

Voor html-links gebruik je de volgende notatie: coderclass
